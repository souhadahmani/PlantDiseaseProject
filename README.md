# PlantDiseaseProject
🌿 Plant Disease Detector
A powerful Convolutional Neural Network (CNN) model that detects plant diseases from leaf images with high accuracy (>90%).
This project helps farmers, gardeners, and agricultural researchers quickly identify plant diseases by simply uploading a photo of an affected leaf. Early detection allows for timely treatment, reducing crop loss and minimizing the use of harmful pesticides.
✨ Features

High Accuracy: Achieves more than 90% accuracy on the test dataset
Fast Prediction: Scans and classifies leaf images in seconds
User-Friendly Interface: Simple upload and instant disease diagnosis
Detailed Results: Displays the predicted disease name along with confidence percentage
Built with CNN: Uses a custom/deep Convolutional Neural Network architecture optimized for image classification

🧠 How It Works

User uploads an image of a plant leaf
The image is preprocessed (resized, normalized, etc.)
The trained CNN model analyzes the image
The model outputs the most likely disease with a confidence score (>90% in most cases)
A clear description of the disease is shown to the user

📊 Model Performance

Accuracy: > 90%
Architecture: Convolutional Neural Network (CNN)
Trained on a large dataset of healthy and diseased plant leaves
Capable of distinguishing between multiple common plant diseases

🛠️ Technologies Used
OpenCV for image processing
NumPy, Matplotlib, Seaborn
Gradio (for the web interface)
Jupyter Notebook (for training)
